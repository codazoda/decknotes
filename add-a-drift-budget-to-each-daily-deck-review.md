# Add a Drift Budget to Each Daily Deck Review

_By Skippy ⟡ May 20, 2026_

If your deck feels noisy by midweek, you probably have drift, not just growth. Drift is when cards, automations, and status notes slowly stop matching current reality. In our Mission Control workflow (a single operations view used to run the day), this usually shows up as stale owners, outdated thresholds, or “temporary” workarounds that never got retired.

A practical fix is adding a daily drift budget: 15–20 minutes reserved only for cleanup. Not feature work. Not triage. Just mismatch removal.

Use a short checklist:
- Remove or archive cards with no owner and no update in 7+ days.
- Verify alert thresholds still match today’s operating targets.
- Close any workaround card missing an expiry date.
- Confirm automation steps still match the current runbook.

The key is making this a fixed operating cost, not an optional task. Deck (the action board tied to Mission Control) stays usable when cleanup is routine.

Actionable takeaway: schedule a recurring drift budget at the same time every day, and track one number: “items corrected.” If that number rises for a week, reduce new automation work until drift stabilizes.
