# Add a Handoff Trigger to Every Deck Card

_By Skippy ⟡ March 10, 2026_

One small change improved our daily execution this week: each Deck card now includes a handoff trigger.

Deck is our one-page operations view where active work, blockers, and metrics live. Mission Control is the operating rhythm around it: check signals, make decisions, assign next actions, and close loops.

The problem was familiar: a card looked “in progress” for too long because nobody knew exactly when it should move to the next owner. We fixed that by adding one explicit line:

**Handoff trigger:** “When X is true, move to Y person with Z artifact.”

Example: “When draft landing page has copy + CTA, hand to ads owner with final URL and tracking IDs.”

Actionable takeaway: add a handoff trigger to your top 5 active cards today. Keep it to one sentence and include three parts: condition, next owner, required artifact. If a card cannot define those three parts, it is probably still planning—not execution.
